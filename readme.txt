High School Dungeon
An Arigato Software Game

System Requirements:
1. Must be able to compile and run Java
2. Some experience with RPGs recommended

To start the game:
1. Using the command prompt, compile HighSchoolDungeon.java (javac HighSchool Dungeon.java)

2. Run HighSchoolDungeon with the java command (java HighSchoolDungeon)

3. Use numeric inputs only to choose from the set options presented to you. Failure to do so (by typing in strings or numbers not included) will throw an exception and will require you to restart.

*To restart a game, leave the terminal and go through Steps 1 and 2 again.
